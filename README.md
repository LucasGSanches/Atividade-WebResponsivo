# Atividade-WebResponsivo
Atividade universitária da disciplina de desenvolvimento web sobre html e css

**Objetivo do projeto** \**
Aprender novos elementos, boas práticas, principalmente sobre responsividade

**Estratégia de responsividade adotada** \**
Para uma boa adequação do site a vários tamanhos de telas foi utilizado css grid e alterações dinâmicas
de suas propriedades pelo uso de Media Queries.

**Breakpoints utilizados** \**

* minimo de 800 px \**
Foi utilizado considerando o contexto de tablets, com modificação para ser disposto dois cards por linha e o 
menu disposto em linha

* minimo de 1280 px \**
Foi utilizado considerando o contexto de tablets, com modificação para ser disposto três cards por linha,
além do menu disposto em linha
